# 版本回滚说明

本目录已使用 Git 管理版本。每次完成一批数据或代码整理后，建议提交一次版本。

## 常用命令

查看当前改动：

```bash
git status
```

提交一个新版本：

```bash
git add .
git commit -m "描述本次修改"
```

查看版本记录：

```bash
git log --oneline
```

回到某个历史版本：

```bash
git restore --source <版本号> -- .
```

如果确认要把整个目录退回到某个版本：

```bash
git reset --hard <版本号>
```
